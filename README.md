# My Hacker News RSS

A Python-based RSS feed generator that filters [Hacker News](https://news.ycombinator.com/) top stories based on customizable keywords.

A feed generated by this tool with only AI-related topics is available at https://ubalklen.github.io/my-hacker-news-rss/feed.xml

## Requirements

- Python 3.12 or higher
- Dependencies managed via `uv` package manager

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ubalklen/my-hacker-news-rss.git
   cd my-hacker-news-rss
   ```

2. Install `uv` (if not already installed):
   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```

3. Install dependencies:
   ```bash
   uv sync
   ```

## Usage

### Running Locally

Generate the RSS feed manually:

```bash
uv run src/main.py
```

The generated RSS feed will be saved to `public/feed.xml`.

## Configuration

### Customizing Keywords
You can provide keywords in three ways:

1. **Command Line Arguments**:
   ```bash
   uv run src/main.py --keywords "Python" "Machine Learning" "Rust"
   ```

2. **Keywords File**:
   ```bash
   uv run src/main.py --keywords-file "keywords.txt"
   ```
   The file should contain one keyword per line. See [`keywords.txt`](keywords.txt) for an example.

3. **Environment Variable**:
   ```bash
   # Linux/macOS
   export MY_KEYWORDS="Python,Machine Learning,Rust"
   uv run src/main.py --keywords-env MY_KEYWORDS

   # Windows (PowerShell)
   $env:MY_KEYWORDS="Python,Machine Learning,Rust"
   uv run src/main.py --keywords-env MY_KEYWORDS
   ```

If no arguments are provided, no filter will be applied.

### Automated Updates

The repository is configured with GitHub Actions to automatically:
- Run every 6 hours (configurable in `.github/workflows/update_feed.yml`)
- Generate a fresh RSS feed
- Deploy it to GitHub Pages

You can also manually trigger the workflow from the Actions tab in GitHub.

## RSS Feed

Once deployed, the RSS feed is available at:
```
https://<username>.github.io/my-hacker-news-rss/feed.xml
```

Replace `<username>` with your GitHub username.
